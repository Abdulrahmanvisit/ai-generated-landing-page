# Tempo — Landing Page

A responsive, one-page marketing site for a fictional productivity app called Tempo, built with semantic HTML5 and CSS (Flexbox + Grid).

## Sections
- Hero
- Features
- Call to action (CTA)
- Footer

## Built with
- Semantic HTML5
- CSS Flexbox and Grid for layout
- A media query for responsive design (mobile breakpoint at 640px)

## AI usage
I used AI to help plan the project: it suggested the app name ("Tempo"), the hero headline and subtext, feature card copy, and the color palette/font pairing. I wrote and typed all the HTML and CSS myself, module by module, testing in the browser after each section.
